#### Args:


* <b>`quantity`</b>: A Tensor of type tf.int32, representing the quantities
  of period types (e.g. how many months). Can be both positive and
  negative.
* <b>`period_type`</b>: A PeriodType (a day, a month, etc). Currently only one
